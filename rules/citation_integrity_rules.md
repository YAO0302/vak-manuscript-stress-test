# Citation Integrity Rules

This rule file defines how citations and references must be handled in Russian-language academic manuscripts prepared for ВАК journals and related scholarly publications.

The purpose is to prevent fabricated references, decorative citations, unsupported claims, and citation misuse.

## Core principle

A citation is not decoration.

A citation must perform a clear scholarly function.

Every reference used in a manuscript should satisfy three conditions:

1. The source is real.
2. The bibliographic data are correct.
3. The source supports the claim it is attached to.

If one of these conditions is not satisfied, the citation is not ready for submission.

## 1. No fabricated references

Never invent:

* authors;
* article titles;
* book titles;
* journal names;
* publishers;
* DOI numbers;
* volume numbers;
* issue numbers;
* page ranges;
* URLs;
* database records.

A plausible-looking reference is not enough.

A source must be verified before submission.

## 2. No fake DOI numbers

DOI numbers must not be guessed.

A DOI is valid only if:

* it resolves correctly;
* it leads to the same article;
* the author names match;
* the title matches;
* the journal or publisher matches;
* the year matches.

High-risk DOI signs:

* DOI does not resolve;
* DOI leads to a different article;
* DOI belongs to another source;
* DOI format looks correct but cannot be found;
* DOI is copied from another reference;
* DOI is generated by AI.

If DOI is uncertain, mark:

```text
[VERIFY DOI]
```

## 3. No decorative citations

Do not add citations only to make the text look scholarly.

A citation is decorative if:

* it appears after a broad claim but does not support it;
* it is placed at the end of a paragraph with many unrelated claims;
* it is not discussed in the text;
* it is included only to increase bibliography size;
* it belongs to another field;
* it does not shape the argument;
* it is never used analytically.

Decorative citations should be removed or replaced with verified relevant sources.

## 4. Citation must support the claim

For every cited claim, ask:

```text
Does this source actually support this exact claim?
```

Classify citation support as:

* directly supported;
* partially supported;
* weakly supported;
* unsupported;
* source unavailable;
* citation mismatch.

If the source supports only part of the claim, narrow the claim.

If the source does not support the claim, remove or replace the citation.

Do not invent replacement sources.

## 5. Match citation type to claim type

Different claims require different kinds of sources.

### Theoretical claim

Needs:

* theory texts;
* conceptual studies;
* scholarly monographs;
* peer-reviewed theoretical articles.

### Historical claim

Needs:

* historical sources;
* archival materials;
* architectural history studies;
* primary documents;
* reliable scholarly works.

### Empirical claim

Needs:

* data;
* observations;
* surveys;
* interviews;
* field studies;
* statistical sources;
* empirical publications.

### Health or active longevity claim

Needs:

* gerontology literature;
* public health studies;
* environmental psychology;
* urban health research;
* empirical evidence;
* carefully defined mechanism.

### Architectural claim

Needs:

* architectural analysis;
* project documentation;
* plans;
* photographs;
* design descriptions;
* architectural theory;
* spatial analysis.

### Cultural interpretation

Needs:

* cultural theory;
* semiotic analysis;
* historical-cultural sources;
* textual or visual evidence;
* relevant scholarly interpretation.

## 6. Avoid overclaiming with citations

A real source can still be misused.

High-risk patterns:

* one case study used as general proof;
* theoretical source used as empirical evidence;
* architectural description used as health evidence;
* cultural symbolism used as proof of social effect;
* historical source used to support a contemporary claim;
* source about one country used to support another country;
* source about one group used to generalize to all groups.

Fix:

* narrow the claim;
* add stronger evidence;
* use a better source;
* move the citation to a more appropriate claim;
* remove the claim if evidence is unavailable.

## 7. In-text citation and bibliography matching

Every in-text citation must have a matching bibliography entry.

Every bibliography entry must be cited in the manuscript.

Check:

* author name consistency;
* year consistency;
* transliteration consistency;
* spelling consistency;
* duplicate entries;
* same author and year markers, such as 2020a and 2020b;
* Russian and English versions of the same source.

Problems to fix:

* in-text citation missing from bibliography;
* bibliography entry never cited;
* year mismatch;
* author mismatch;
* title mismatch;
* duplicate references;
* inconsistent transliteration.

## 8. Russian-language source handling

For Russian sources, check:

* author initials;
* title capitalization;
* journal title;
* year;
* issue number;
* page range;
* DOI, if available;
* eLIBRARY or CyberLeninka record, if available;
* РИНЦ information, if relevant;
* official journal page.

Do not translate Russian titles unless the journal requires it.

If transliteration is required, use one consistent system.

## 9. International source handling

For international sources, check:

* author names;
* article title;
* journal title;
* year;
* volume;
* issue;
* page range or article number;
* DOI;
* publisher page;
* Crossref record;
* Scopus or Web of Science record, if available.

Do not translate titles unless the journal requires it.

## 10. Unverifiable sources

If a source cannot be verified, mark it clearly.

Use:

```text
[UNVERIFIABLE SOURCE]
```

If only part of the source is uncertain, use:

```text
[VERIFY AUTHOR]
[VERIFY TITLE]
[VERIFY YEAR]
[VERIFY JOURNAL]
[VERIFY PAGES]
[VERIFY DOI]
[VERIFY URL]
```

Do not silently correct uncertain references.

Do not invent missing bibliographic elements.

## 11. AI-generated reference risk

AI-generated references often look plausible but may be false.

Warning signs:

* generic academic title;
* real author with fake article;
* real journal with fake article;
* DOI does not resolve;
* volume and issue do not match journal history;
* article cannot be found in databases;
* title appears nowhere online;
* page range looks artificial;
* references have repeated artificial patterns;
* source is too perfectly aligned with the user's claim.

If a reference shows these signs, treat it as suspicious until verified.

## 12. Claim risk levels

### Low-risk claim

A narrow descriptive claim supported by visible material.

Example:

```text
The building contains decorative elements associated with traditional motifs.
```

### Medium-risk claim

An interpretive claim requiring theory and analysis.

Example:

```text
These elements function as signs of cultural memory.
```

### High-risk claim

A broad comparative, causal, social, or health-related claim.

Example:

```text
Architectural space supports active longevity.
```

### Critical-risk claim

A claim with no source, fake source, or evidence mismatch.

Example:

```text
Traditional architectural symbols directly improve longevity.
```

Critical-risk claims must be revised before submission.

## 13. Active longevity citation warning

Claims about active longevity require especially careful citation.

A cultural or architectural source may support claims about:

* spatial organization;
* symbolic meaning;
* cultural memory;
* environmental affordances;
* social interaction;
* mobility;
* everyday practices.

It usually cannot support direct claims about:

* medical outcomes;
* biological aging;
* life expectancy;
* health improvement;
* causal effects on longevity.

If the manuscript does not contain empirical health evidence, avoid direct medical claims.

Use mechanism-based, field-appropriate claims instead.

Weak claim:

```text
Traditional Chinese architectural elements improve active longevity.
```

Stronger claim:

```text
Traditional Chinese architectural elements can be analyzed as part of an environment that supports social interaction, mobility, orientation, and everyday practices associated with active longevity.
```

## 14. Citation repair rules

When a citation problem is found, use one of these actions:

* verify the source;
* correct bibliographic data using reliable evidence;
* narrow the claim;
* add a verified relevant source;
* move the citation;
* remove the citation;
* remove the unsupported claim;
* mark the issue for author verification.

Do not:

* invent a replacement source;
* fabricate DOI;
* attach an unrelated citation;
* hide the problem with cautious wording;
* keep a source because it looks scholarly.

## 15. Pre-submission citation checklist

Before submission, check:

* Every source is real.
* Every DOI is correct.
* Every in-text citation has a bibliography entry.
* Every bibliography entry is cited.
* Every citation supports the attached claim.
* No source is used as decoration.
* No unverifiable source remains unmarked.
* No fabricated reference remains.
* No broad claim relies on a weak source.
* No health or longevity claim lacks field-appropriate support.
* No transliteration inconsistencies remain.
* No duplicate references remain.

## Final rule

If a citation does not support the claim, the solution is not to make the sentence more cautious.

The solution is to fix the claim, fix the evidence, or remove the citation.
