# Conclusion Template

This template helps write conclusions for Russian-language academic articles prepared for ВАК journals or related scholarly publications.

The conclusion should complete the article's logic.

It should return to the research aim, summarize supported findings, and state the scholarly contribution without exceeding the evidence.

## Purpose

A strong conclusion should answer three questions:

1. What did the article establish?
2. How does this answer the research question?
3. What is the scholarly contribution?

The conclusion should not merely repeat the abstract.

It should not introduce new sources, new concepts, new examples, or unsupported claims.

## Basic rule

Do not claim more than the article has shown.

Do not invent results.

Do not add broad generalizations without evidence.

Do not use defensive over-explanation.

If the result is not supported by the manuscript, mark it as:

[RESULT REQUIRES EVIDENCE]

If a claim requires a source, mark it as:

[SOURCE REQUIRED]

## Recommended conclusion structure

Use the following order unless the target journal profile requires another structure:

1. Return to the aim of the article.
2. State the main result.
3. Summarize key findings.
4. Explain how the findings answer the research question.
5. State the scholarly contribution.
6. Indicate practical or theoretical significance, if justified.
7. Mention future research only if appropriate.

## 1. Return to the aim

Function:

Remind the reader what the article set out to do.

Recommended Russian formulas:

* Проведенный анализ был направлен на...
* Цель статьи состояла в...
* В рамках исследования был рассмотрен...
* В статье были проанализированы...

Template:

```text
Проведенный анализ был направлен на [aim].
```

Example:

```text
Проведенный анализ был направлен на выявление способов трансформации элементов традиционной китайской культуры в современной архитектурной среде.
```

## 2. Main result

Function:

State the main supported finding.

Recommended Russian formulas:

* В результате исследования установлено...
* Проведенный анализ показал...
* Анализ позволил выявить...
* Полученные результаты свидетельствуют о...
* В ходе исследования были определены...

Template:

```text
Проведенный анализ показал, что [main supported result].
```

Example:

```text
Проведенный анализ показал, что элементы традиционной китайской культуры в современной архитектуре функционируют не только как декоративные мотивы, но и как пространственные, символические и средовые компоненты.
```

If this result is not supported by the article, mark:

```text
[RESULT REQUIRES EVIDENCE]
```

## 3. Key findings

Function:

Summarize the main findings in a clear sequence.

Recommended Russian formulas:

* Во-первых, было выявлено...
* Во-вторых, установлено...
* В-третьих, показано...
* Кроме того, анализ позволил определить...

Template:

```text
Во-первых, [finding 1]. Во-вторых, [finding 2]. В-третьих, [finding 3].
```

Example:

```text
Во-первых, было выявлено, что традиционные культурные элементы сохраняют связь с исторической памятью через визуальные и символические формы. Во-вторых, установлено, что в современной архитектурной среде эти элементы приобретают новые функции, связанные с организацией пространства и повседневных практик. В-третьих, показано, что их значение для активного долголетия может быть рассмотрено через параметры социальной активности, телесной подвижности и восприятия среды.
```

## 4. Answer to the research question

Function:

Show how the findings answer the research question.

Recommended Russian formulas:

* Таким образом, поставленный исследовательский вопрос получает следующее решение...
* Это позволяет ответить на исследовательский вопрос следующим образом...
* Полученные результаты позволяют утверждать...
* В связи с этим можно заключить...

Template:

```text
Таким образом, [research question] может быть раскрыт через [answer based on findings].
```

Example:

```text
Таким образом, трансформация традиционных культурных элементов в современной архитектуре может быть раскрыта через изменение их функции: от воспроизведения исторических форм к участию в организации среды, поддерживающей социальное взаимодействие и устойчивые повседневные практики.
```

## 5. Scholarly contribution

Function:

State what the article adds to the field.

Possible contribution types:

* conceptual clarification;
* methodological clarification;
* new interpretation of material;
* new classification;
* connection between previously separated fields;
* refinement of an existing concept;
* application of a method to new material.

Recommended Russian formulas:

* Научная значимость исследования заключается в...
* Теоретическая значимость статьи состоит в...
* Полученные результаты позволяют уточнить...
* Статья расширяет представления о...
* Предложенный подход позволяет...

Template:

```text
Научная значимость исследования заключается в [specific contribution].
```

Example:

```text
Научная значимость исследования заключается в уточнении подхода к анализу традиционных культурных элементов как компонентов архитектурной среды, связанных с культурной памятью, пространственной организацией и практиками активного долголетия.
```

## 6. Practical or theoretical significance

Function:

Mention significance only if it is justified by the article.

Recommended Russian formulas:

* Практическая значимость результатов связана с...
* Полученные результаты могут быть использованы...
* Теоретическая значимость исследования состоит в...
* Материалы статьи могут быть полезны для...

Template:

```text
Практическая значимость результатов связана с [justified application].
```

Avoid claiming application if the article did not provide evidence for it.

Weak version:

```text
Результаты исследования могут полностью решить проблему активного долголетия.
```

Better version:

```text
Полученные результаты могут быть использованы при дальнейшем изучении архитектурных параметров среды, поддерживающих социальную активность и повседневную мобильность пожилых людей.
```

## 7. Future research

Function:

Mention future research only when it follows naturally from the article.

Recommended Russian formulas:

* Дальнейшее изучение данной проблемы может быть связано с...
* Перспективным направлением дальнейших исследований является...
* В дальнейшем представляется целесообразным рассмотреть...

Template:

```text
Дальнейшее изучение данной проблемы может быть связано с [future research direction].
```

Do not use future research to compensate for weak current evidence.

## Full conclusion skeleton

```text
Проведенный анализ был направлен на [aim]. В результате исследования установлено, что [main supported result].

Во-первых, [finding 1]. Во-вторых, [finding 2]. В-третьих, [finding 3].

Таким образом, [answer to research question]. Научная значимость исследования заключается в [specific contribution].

[Optional practical/theoretical significance]. [Optional future research direction].
```

## Conclusion for architecture and cultural studies

```text
Проведенный анализ был направлен на выявление способов трансформации элементов традиционной китайской культуры в современной архитектурной среде. В результате исследования установлено, что данные элементы функционируют не только как декоративные мотивы, но и как пространственные, символические и средовые компоненты.

Во-первых, было выявлено, что традиционные культурные элементы сохраняют связь с исторической памятью через визуальные и символические формы. Во-вторых, установлено, что в современной архитектурной среде они приобретают новые функции, связанные с организацией пространства и повседневных практик. В-третьих, показано, что их значение для активного долголетия может быть рассмотрено через параметры социальной активности, телесной подвижности и восприятия среды.

Таким образом, трансформация традиционных культурных элементов в современной архитектуре может быть раскрыта через изменение их функции: от воспроизведения исторических форм к участию в организации среды, поддерживающей социальное взаимодействие и устойчивые повседневные практики. Научная значимость исследования заключается в уточнении подхода к анализу традиционных культурных элементов как компонентов архитектурной среды, связанных с культурной памятью, пространственной организацией и практиками активного долголетия.

Дальнейшее изучение данной проблемы может быть связано с анализом конкретных архитектурных объектов и проверкой связи пространственных параметров среды с повседневными практиками различных возрастных групп.
```

## Common problems

### Problem 1. Conclusion only repeats the abstract

Weak:

```text
В статье была рассмотрена актуальная проблема трансформации традиционной китайской культуры в современной архитектуре.
```

Better:

```text
Проведенный анализ позволил выявить основные способы трансформации традиционных культурных элементов: сохранение символической формы, изменение пространственной функции и включение в сценарии повседневного взаимодействия.
```

### Problem 2. Conclusion exceeds evidence

Weak:

```text
Традиционная китайская архитектура доказывает возможность обеспечения активного долголетия.
```

Better:

```text
Связь архитектурной среды с активным долголетием может быть рассмотрена через параметры, влияющие на социальную активность, телесную подвижность и устойчивость повседневных практик.
```

### Problem 3. New concepts appear in conclusion

Weak:

```text
Таким образом, проблема может быть объяснена через биофильный дизайн и нейроархитектуру.
```

Better:

```text
Таким образом, выводы должны быть сформулированы через те понятия и методы, которые уже использовались в статье.
```

### Problem 4. Defensive conclusion

Weak:

```text
Настоящая статья не претендует на исчерпывающее решение проблемы.
```

Better:

```text
Дальнейшее изучение данной проблемы может быть связано с расширением корпуса анализируемых архитектурных объектов и уточнением критериев оценки средовых параметров.
```

## Final conclusion checklist

Before using the conclusion, check:

* Does it return to the aim?
* Does it answer the research question?
* Are all results supported by the article?
* Does it summarize findings in a clear sequence?
* Does it state contribution precisely?
* Does it avoid new sources?
* Does it avoid new concepts?
* Does it avoid unsupported broad claims?
* Does it avoid defensive over-explanation?
* Does it avoid simply repeating the abstract?
* Does it fit the target journal style profile?
