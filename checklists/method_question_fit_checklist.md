# Method-Question Fit Checklist

This checklist is used to test whether the research question, method, material, analysis, and conclusion fit each other.

It is designed for Russian-language academic manuscripts prepared for ВАК journals and related scholarly publications.

## Core rule

A method is acceptable only if it can answer the research question.

A method is not acceptable simply because it sounds academic.

If the method cannot support the claim, revise one of the following:

* research question;
* aim;
* material;
* method;
* analysis;
* conclusion.

Do not hide a weak method behind cautious or defensive wording.

## 1. Research question check

The research question should be explicit, narrow, and answerable.

Check:

* Is the research question clearly stated?
* Is it narrower than the general topic?
* Can it be answered in one article?
* Does it require the material provided?
* Does it require the method selected?
* Is the conclusion actually answering this question?

Weak question:

```text
How does traditional Chinese culture influence active longevity?
```

Stronger question:

```text
How are elements of traditional Chinese culture transformed in contemporary architectural environments associated with active longevity practices?
```

Status:

```text
[ ] Fits the article
[ ] Too broad
[ ] Not explicit
[ ] Does not match the method
[ ] Needs revision
```

## 2. Aim check

The aim should follow from the research question.

Check:

* Does the aim answer the research question?
* Is the aim achievable with the manuscript's material?
* Does the aim use a clear analytical verb?
* Does the aim avoid unsupported causal promises?
* Does the conclusion return to the aim?

Useful verbs:

* identify;
* analyze;
* compare;
* classify;
* interpret;
* clarify;
* reveal;
* determine.

Weak aim:

```text
The aim is to study traditional culture and active longevity.
```

Stronger aim:

```text
The aim is to identify the ways in which traditional Chinese cultural elements are transformed in contemporary architectural environments and to determine how these elements participate in shaping spatial conditions associated with active longevity practices.
```

Status:

```text
[ ] Fits the research question
[ ] Too broad
[ ] Too vague
[ ] Promises unsupported results
[ ] Needs revision
```

## 3. Object and subject check

The object is the broader phenomenon.

The subject is the specific aspect analyzed.

Check:

* Is the object broader than the subject?
* Is the subject specific?
* Can the subject be analyzed with the selected method?
* Does the subject match the conclusion?
* Does the subject avoid unsupported causal claims?

Example:

```text
Object:
Contemporary architectural environment of China.

Subject:
Transformation of traditional Chinese cultural elements in contemporary architectural space in relation to spatial organization, social interaction, mobility, and everyday practices.
```

Status:

```text
[ ] Object and subject are consistent
[ ] Object and subject are confused
[ ] Subject is too broad
[ ] Subject does not match method
[ ] Needs revision
```

## 4. Material check

The material is what the author actually analyzes.

Check:

* Is the material named?
* Is the material specific?
* Are selection criteria explained?
* Is the material sufficient for the research question?
* Does the material allow the method to be applied?
* Is the material strong enough for the conclusion?

Possible material:

* architectural objects;
* public spaces;
* design projects;
* planning documents;
* photographs;
* spatial plans;
* theoretical texts;
* interviews;
* statistical data;
* policy documents;
* previous studies.

If the material is missing, mark:

```text
[AUTHOR MUST PROVIDE MATERIAL]
```

Status:

```text
[ ] Material fits the question
[ ] Material is incomplete
[ ] Selection criteria missing
[ ] Material too narrow for conclusion
[ ] Needs author verification
```

## 5. Method check

The method must explain how the research question is answered.

Check:

* Is the method named?
* Is the method explained?
* Is each method connected to the question?
* Is each method connected to the material?
* Is each method visible in the analysis?
* Does the method produce the type of result claimed?

Weak method statement:

```text
The article uses comparative, semiotic, and historical-cultural methods.
```

Stronger method statement:

```text
Comparative analysis is used to identify differences between traditional and contemporary architectural forms; semiotic analysis makes it possible to interpret cultural elements as signs; historical-cultural analysis connects these signs to cultural memory and continuity.
```

Status:

```text
[ ] Method fits the question
[ ] Method only listed
[ ] Method not applied
[ ] Method cannot support conclusion
[ ] Needs rewrite
```

## 6. Method function check

Each method must have a specific function.

Use this table:

| Method                       | Function                               | Material | Output |
| ---------------------------- | -------------------------------------- | -------- | ------ |
| Comparative analysis         | Compares cases or forms                |          |        |
| Semiotic analysis            | Interprets signs and meanings          |          |        |
| Historical-cultural analysis | Connects elements to cultural memory   |          |        |
| Spatial analysis             | Examines organization of space         |          |        |
| Discourse analysis           | Analyzes texts or statements           |          |        |
| Content analysis             | Classifies textual or visual material  |          |        |
| Case study                   | Examines one or several cases in depth |          |        |

If a method has no function, remove it or explain it.

## 7. Claim type check

Different claims require different methods and evidence.

### Descriptive claim

Example:

```text
The building contains traditional decorative elements.
```

Suitable support:

* photographs;
* plans;
* architectural descriptions;
* project documentation.

### Interpretive claim

Example:

```text
These elements function as signs of cultural memory.
```

Suitable support:

* semiotic analysis;
* cultural theory;
* historical-cultural sources.

### Comparative claim

Example:

```text
The contemporary form transforms the traditional element.
```

Suitable support:

* comparable traditional and contemporary examples;
* comparison criteria;
* historical or visual evidence.

### Social practice claim

Example:

```text
The spatial organization supports everyday interaction.
```

Suitable support:

* spatial analysis;
* observation;
* architectural theory;
* urban studies sources;
* social practice literature.

### Health or active longevity claim

Example:

```text
The environment supports active longevity practices.
```

Suitable support:

* gerontology;
* environmental psychology;
* public health;
* age-friendly environment research;
* evidence about social interaction, mobility, or daily practices.

Critical warning:

A cultural or semiotic method alone cannot prove direct health outcomes.

## 8. Active longevity fit check

Claims about active longevity are high-risk.

Check:

* Does the manuscript define active longevity?
* Does it explain which aspect of active longevity is studied?
* Does it avoid direct medical or biological claims?
* Does it use relevant sources?
* Does it connect architecture to active longevity through a clear mechanism?

Acceptable mechanism:

```text
architectural environment
↓
spatial organization
↓
mobility / orientation / social interaction
↓
everyday practices
↓
active longevity context
```

High-risk claim:

```text
Traditional architectural elements improve longevity.
```

Stronger claim:

```text
Traditional cultural elements can be analyzed as part of an architectural environment that supports social interaction, mobility, orientation, and everyday practices associated with active longevity.
```

Status:

```text
[ ] Claims are properly limited
[ ] Evidence required
[ ] Claims are too causal
[ ] Health claims unsupported
[ ] Needs revision
```

## 9. Evidence-method fit check

For each major claim, fill this table.

| Claim | Method used | Evidence/material | Does method support claim? | Required action |
| ----- | ----------- | ----------------- | -------------------------- | --------------- |
|       |             |                   | Yes / Partly / No          |                 |
|       |             |                   | Yes / Partly / No          |                 |
|       |             |                   | Yes / Partly / No          |                 |

If the answer is "Partly" or "No", revise the claim or strengthen the evidence.

## 10. Analysis check

The analysis should show the method working on the material.

Check:

* Does the analysis use the stated method?
* Does each example support a claim?
* Does the text explain what the material shows?
* Does the analysis move beyond description?
* Does the analysis connect back to the research question?

Weak analysis:

```text
The building uses traditional elements. This shows the importance of culture.
```

Stronger analysis:

```text
The traditional element functions not only as a visual reference, but also as a spatial marker that organizes movement, orientation, or zones of social interaction. This allows the element to be interpreted as part of an environment connected to everyday practices.
```

Status:

```text
[ ] Analysis fits method
[ ] Analysis is mostly descriptive
[ ] Method is invisible
[ ] Evidence missing
[ ] Needs reconstruction
```

## 11. Conclusion fit check

The conclusion must not exceed the method.

Check:

* Does the conclusion answer the research question?
* Does it follow from the analysis?
* Does it avoid unsupported causal claims?
* Does it avoid new concepts?
* Does it avoid new evidence?
* Does it match the method's limits?

Weak conclusion:

```text
Traditional Chinese architecture proves the possibility of active longevity.
```

Stronger conclusion:

```text
The analysis shows that traditional Chinese cultural elements may participate in shaping architectural environments connected to social interaction, mobility, orientation, and everyday practices associated with active longevity.
```

Status:

```text
[ ] Conclusion fits method and evidence
[ ] Conclusion too broad
[ ] Conclusion introduces new claims
[ ] Conclusion exceeds evidence
[ ] Needs rewrite
```

## 12. Common mismatch patterns

### Pattern 1. Broad question, narrow material

Problem:

The article asks about all contemporary architecture but analyzes only one case.

Fix:

Narrow the question or expand the material.

### Pattern 2. Interpretive method, causal conclusion

Problem:

The article uses cultural or semiotic analysis but concludes with health or social causality.

Fix:

Narrow the conclusion or add appropriate empirical evidence.

### Pattern 3. Method listed but not used

Problem:

The method appears in the introduction but disappears from the analysis.

Fix:

Show how the method operates in each analytical section.

### Pattern 4. Material not defined

Problem:

The article discusses examples but does not define the research material.

Fix:

Add a material section with selection criteria.

### Pattern 5. Conclusion answers a different question

Problem:

The article asks one question but concludes with another issue.

Fix:

Rewrite the conclusion or revise the research question.

## 13. Repair decision

Choose one repair route.

### Route A. Narrow the research question

Use when the question is too broad.

### Route B. Strengthen the material

Use when the material is too weak for the claim.

### Route C. Rewrite the methodology

Use when methods are listed but not operational.

### Route D. Narrow the conclusion

Use when the conclusion exceeds evidence.

### Route E. Add field-appropriate sources

Use when high-risk claims require better support.

### Route F. Rebuild the structure

Use when several elements do not fit each other.

## 14. Final fit decision

Use one of the following decisions.

```text
Fit passed
```

Use only when research question, method, material, analysis, and conclusion are aligned.

```text
Minor revision required
```

Use when the method fits the question but needs clearer explanation.

```text
Major revision required
```

Use when the question, method, material, or conclusion do not fully match.

```text
Do not submit
```

Use when the conclusion makes claims the method and evidence cannot support.

## Final checklist

Before submission, confirm:

* research question is explicit;
* aim follows from the question;
* object and subject are consistent;
* material is defined;
* selection criteria are explained;
* method fits the question;
* method fits the material;
* method is applied in the analysis;
* each major claim has evidence;
* conclusion does not exceed method;
* active longevity claims are properly supported or narrowed;
* no weak logic is hidden by defensive wording.

## Final rule

The strongest manuscript is not the one with the most methods.

The strongest manuscript is the one where the method actually answers the question.
